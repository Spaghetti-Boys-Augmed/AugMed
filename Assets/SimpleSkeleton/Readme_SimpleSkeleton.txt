Simple Skeleton is a humanoid-compliant character model. 
The model has 26771 vertices, 34264 polygons, and 2k texture size. 
The following animations are included with this model.

 Waiting
 Walking (straight, curve)
 Back walking
 Running (straight, curve)
 Jumping
 Rolling
 Lying down
 Fighting pose
 Punching (Right / Left)
 Change of direction
 Knockdown

A controller that plays these animations is also included, and you can immediately experience this character in the demo scene after downloading.
